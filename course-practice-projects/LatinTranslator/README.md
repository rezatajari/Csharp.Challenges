# Latin Translator (Windows Form)

## Concept

This challenge focuses on **Windows Forms application development**, user interaction, and event handling.  
You’ll build a simple translator tool that demonstrates how GUI elements (buttons, labels) interact with code to provide meaningful feedback to users.

## The Challenge

Create a Windows Form application and:

1. Add **three buttons**, each representing a Latin word:
   - sinister  
   - dexter  
   - medium  
2. Add a **Label** control to display the translation.
3. When a user clicks a button, the label should display its corresponding English translation:

| Latin    | English |
|----------|---------|
| sinister | left    |
| dexter   | right   |
| medium   | center  |

4. Ensure the application updates the label correctly for each click.

### Example Behavior

- User clicks **sinister** → Label shows **left**  
- User clicks **dexter** → Label shows **right**  
- User clicks **medium** → Label shows **center**

## Why This Matters

This exercise teaches you how to:

- Work with **Windows Forms controls** (Button, Label)  
- Handle **button click events**  
- Connect **UI elements** to program logic  
- Provide **interactive feedback** in a desktop application

# مترجم لاتین (Windows Form)

## مفهوم

این چالش بر توسعهٔ برنامه‌های **ویندوزفرم**، تعامل کاربر و مدیریت رویدادها تمرکز دارد.  
در این تمرین، یک ابزار مترجم ساده می‌سازید که نشان می‌دهد عناصر گرافیکی (دکمه و لیبل) چگونه با کد در ارتباط هستند و به کاربر بازخورد ارائه می‌دهند.

---

## چالش

برنامه باید یک Windows Form باشد و:

۱. سه دکمه اضافه کند که هرکدام نمایندهٔ یک واژهٔ لاتین هستند:

- sinister  
- dexter  
- medium  

۲. یک **Label** برای نمایش ترجمه قرار دهد.  

۳. با کلیک هر دکمه، ترجمهٔ انگلیسی آن واژه نمایش داده شود:

| لاتین     | انگلیسی |
|-----------|---------|
| sinister  | left    |
| dexter    | right   |
| medium    | center  |

۴. برنامه باید به‌درستی و سریع متن لیبل را با هر کلیک به‌روزرسانی کند.

---

## نمونه رفتار

- کاربر روی **sinister** کلیک کند → لیبل **left** را نمایش می‌دهد  
- کاربر روی **dexter** کلیک کند → لیبل **right** را نمایش می‌دهد  
- کاربر روی **medium** کلیک کند → لیبل **center** را نمایش می‌دهد  

---

## چرا این تمرین مهم است؟

با این چالش یاد می‌گیرید:

- چگونه با کنترل‌های ویندوزفرم مانند **Button** و **Label** کار کنید  
- چگونه رویدادهای **Click** را مدیریت کنید  
- چگونه منطق برنامه را به عناصر گرافیکی متصل کنید  
- چگونه بازخورد تعاملی و سریع برای کاربر فراهم کنید
