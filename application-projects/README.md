# Application Projects 

This directory contains more complete and structured C# applications compared to basic coding challenges. These projects focus on applying object-oriented programming, application architecture, and real-world scenarios.

## Purpose

The goal of this section is to move beyond small exercises and practice building full applications with:

- Object-Oriented Design (OOP)
- Separation of concerns
- File or data persistence
- API-based architecture (in some projects)
- Real-world problem solving

## Projects

### 1. TodoApp
A task management application for creating, updating, and tracking daily tasks.

Key concepts:
- CRUD operations
- Task status management
- Basic architecture design

---

### 2. HabitTracker
An application for tracking daily habits and monitoring consistency over time.

Key concepts:
- Data tracking over time
- Simple analytics logic
- Persistent storage

---

### 3. FinanceTracker
A personal finance management system for tracking income and expenses.

Key concepts:
- Financial data modeling
- Transaction categorization
- Basic reporting logic

---

## Notes

These projects are part of a learning journey from basic C# exercises toward more structured software development practices. The focus is not just on functionality but also on design clarity and maintainability.
