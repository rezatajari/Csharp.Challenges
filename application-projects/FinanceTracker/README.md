# C# 12 Engineering Journey

This repository documents a structured, project-based mastery path through C# 12 and .NET 8.

Based on:
Programming C# 12 (Ian Griffiths)

## Objectives

- Deep understanding of C# language features
- Runtime model (CLR, IL, JIT, AOT)
- Advanced LINQ and functional patterns
- Async and concurrency mastery
- Reflection and metadata
- Assemblies and versioning
- Performance and memory optimization

## Architecture Evolution

The project evolves through multiple phases:

1. Queryable Expense Engine
2. Event-Driven Accounting Core
3. Async Reporting Service
4. Reflection-Based Rule Engine
5. Multi-Targeted Library Packaging
6. Performance & Memory Engineering

Each phase builds on the previous one.
