# for-self

### Overview
The **for-self** folder is a personal practice space inside the main repository.  
It is dedicated to exercises that I design, experience, and implement myself based on real learning needs and hands-on encounters.

This folder contains **self-driven, experimental practices** focused mainly on **C# fundamentals** and common scenarios that arise during real development work.

The goal of this folder is not perfection or completeness, but **learning through direct experience**.

### Purpose
- Practice C# core concepts through personal experiments  
- Reinforce understanding by implementing ideas from scratch  
- Explore edge cases and real-world problems encountered during coding  
- Keep a record of learning progress and thought processes  

### What You’ll Find Here
- Small, focused C# exercises  
- Experimental code snippets  
- Practice projects inspired by real problems  
- Notes and implementations around fundamental concepts  

This folder represents **learning by doing**, mistakes included.
